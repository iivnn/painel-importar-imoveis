Casa local app

Centralize anuncios de aluguel e compare oportunidades com SWOT.

Template inicial em Angular + .NET para cadastrar imoveis, categorizar, visualizar localizacao em mapa e registrar analises.

Primeira meta

  • CRUD de anuncios
  • Categorias e status
  • Endereco + coordenadas
  • Analise SWOT por imovel

Frontend

Angular standalone com rotas, componentes e servicos para consumir a API local.

Backend

ASP.NET Core Web API preparada para evoluir com SQLite, EF Core e endpoints REST.

Mapa

Espaco reservado para integrar Leaflet e exibir os imoveis por localizacao.

Proximo passo sugerido

Criar a entidade Anuncio no backend e a tela de listagem no Angular.
